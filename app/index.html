Tananã

♫ ♫ ♫ ♫ ♫

Tananã é um software educativo para aprender a ler partituras. O formato de arquivo usado é o MusicXML, um padrão usado mundialmente para partituras musicais. Aqui, por exemplo, podem ser encontrados programas que sabem ler/escrever o formato MusicXML.

Se você está em dúvida, abra o exemplo inicial para ver como funciona

Ou escolha uma pasta para servir de biblioteca:

Tananã vai ler esta pasta, suas subpastas e arquivos em busca de músicas em xml ;)

    Este projeto é open source e é mantido por Matheus Graciano
